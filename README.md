# AI Impact on Jobs 2030

## Project Overview
This project analyzes the impact of Artificial Intelligence on jobs, salaries, automation risk, and future workforce demand by 2030.

## Dataset
- 3,000 Records
- 20 Features
- 10 Industries
- 10 Countries

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- Excel

## Key Analysis
- Data Cleaning
- Statistical Analysis
- Correlation Analysis
- Salary Trends
- AI Replacement Risk
- Future Demand Analysis

- Executive Dashboard
- Deep Dive Analytics Dashboard

## Visualizations
